## web-project
CS 456 – Web Programming (Spring 2026)

### About
Updraft GG is a video game distribution platform. It will operate as a direct competitor to Valve's Steam. It should include functionality for users to browse and purchase games, download additional content like mods, and interact with other users through friends lists, comments, and activity status. The target audience encompasses competitive gamers, casual gamers, modding community, game developers and game publishers. Developers and publishers should have more functionalities than regular users including managing their games/storefronts, accessing marketing and financial data, and controlling DRM.

### Run
- Install dependencies with `npm install`
- Start the server with `npm run dev`
